Bring your Wacom Intuos Pro Small back to life.

Wacom no longer ships a driver for the PTH-451 on recent macOS releases. wacomd is a userspace open-source driver that restores pen input, 2048-level pressure, eraser detection and multi-finger touch gestures — no kernel extension, no Wacom official driver needed.

What works

Tested live on PTH-451 / macOS 26.3 / Apple Silicon. ~200 events/s on the pen, matching the Wacom HID native rate.

Absolute pen

1:1 position mapping to the primary screen, linear 11-bit pressure curve read by every app that consumes NSEvent.tabletPoint.

Automatic eraser

Detected from the toolID in the Wacom proximity packet. Photoshop switches to the eraser tool the moment you flip the pen.

Trackpad mode

The touch surface drives the cursor in relative trackpad mode, with non-linear acceleration curve and sub-pixel accumulation.

2-finger scroll

Smoothed centroid, sub-pixel accumulator, sensitivity adjustable from 0.05 to 2.0 px per tablet unit. "Natural / classic" direction toggle.

3-finger gestures

Vertical swipe = Mission Control / Exposé. Horizontal swipe = switch Spaces. Tap = middle click.

Coexists with Wacom

If the official Wacom driver is still installed, a script disables its daemons without removing anything — fully reversible.

Technical overview

For the curious and for anyone willing to port wacomd to other Wacom models.

Architecture

Swift userspace daemon using IOHIDManager to read the tablet (vendor page 0xff0d for the pen, page 0xff00 for touch), and CGEvent to inject clicks, cursor moves and .tabletProximity / .tabletPoint events carrying pressure. No kernel extension, no special Apple entitlement — every sensitive capability is gated by TCC at runtime.

Pen protocol

Intuos5/Pro format reverse-engineered from the Linux driver (drivers/hid/wacom_wac.c). 10-byte report, Report ID 2:

  • X = (b[2]<<9) | (b[3]<<1) | (b[9]>>1)&1
  • Y = (b[4]<<9) | (b[5]<<1) | b[9]&1
  • 11-bit pressure in b[6..7] + LSB inside the status byte
  • Proximity enter/leave on Report ID 0xc0

Touch protocol

Multi-touch surface in 12-bit asymmetric packed format: each slot (up to 5) takes 8 bytes, the high-order byte holds X or Y, the high nibble of a neighbouring byte completes the remaining 4 bits. The firmware does not expose the ExpressKeys nor the Touch Ring over HID — they remain unreachable without direct USB access (IOUSBLib).

Configuration

JSON file at ~/Library/Application Support/wacomd/config.json. The daemon hot-reloads it on receipt of SIGHUP, sent automatically by the configurator app on every change. No service restart needed.

Compatibility

System

  • macOS 26 Tahoe — tested
  • macOS 15 Sequoia — should work (public APIs unchanged)
  • macOS 14 Sonoma — same
  • Apple Silicon + Intel

Tablets

  • Wacom Intuos Pro Small — PTH-451 (VID 0x056a / PID 0x0314) ✓ tested
  • Intuos5 / Pro family — untested but identical report format under Linux, one-line addition to KnownModels.swift
  • Bluetooth models — not supported yet
