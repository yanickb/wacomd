Ramène ta Wacom Intuos Pro Small à la vie.

Wacom n'écrit plus de pilote pour la PTH-451 sur les macOS récents. wacomd est un driver userspace open-source qui rétablit stylet, pression 2048 niveaux, gomme et gestes tactiles multi-doigts — sans extension noyau, sans driver Wacom officiel.

Ce qui fonctionne

Testé en live sur PTH-451 / macOS 26.3 / Apple Silicon. ~200 évènements / s sur le stylet, identique au taux natif du HID Wacom.

Stylet absolu

Position 1:1 sur l'écran principal, courbe de pression linéaire 11-bit reconnue par toutes les apps qui lisent NSEvent.tabletPoint.

Gomme automatique

Détectée via le toolID dans le paquet de proximité Wacom. Photoshop bascule sur l'outil gomme dès que tu retournes le stylet.

Trackpad mode

La surface tactile pilote le curseur en relatif comme un trackpad MacBook, avec courbe d'accélération non-linéaire et sub-pixel.

Scroll 2 doigts

Centroïde lissé, sub-pixel accumulator, sensibilité ajustable de 0.05 à 2.0 px par unité tablette. Inversion "naturel / classique".

Gestes 3 doigts

Swipe vertical = Mission Control / Exposé. Swipe horizontal = switch Spaces. Tap = middle click.

Coexiste avec Wacom

Si le driver Wacom officiel est encore installé, un script désactive ses daemons sans rien supprimer — réversible.

Présentation technique

Pour les curieux et les gens qui voudraient porter wacomd vers d'autres modèles Wacom.

Architecture

Démon Swift en userspace utilisant IOHIDManager pour lire la tablette (page vendor 0xff0d pour le stylet, page 0xff00 pour le tactile), et CGEvent pour injecter clics, mouvements de curseur et évènements .tabletProximity / .tabletPoint avec pression. Aucune extension noyau, aucun entitlement Apple spécifique — toutes les capabilities sensibles sont gérées par TCC au runtime.

Protocole stylet

Reverse-engineering du format Intuos5/Pro depuis le driver Linux (drivers/hid/wacom_wac.c). Rapport 10 octets, Report ID 2 :

  • X = (b[2]<<9) | (b[3]<<1) | (b[9]>>1)&1
  • Y = (b[4]<<9) | (b[5]<<1) | b[9]&1
  • Pression 11 bits sur b[6..7] + LSB dans le status byte
  • Proximité enter/leave sur Report ID 0xc0

Protocole tactile

Surface multi-touch en 12-bit packé asymétrique : chaque slot (5 max) occupe 8 octets, l'octet de poids fort donne X ou Y, le nibble haut d'un octet voisin complète les 4 bits restants. Le firmware n'expose pas les ExpressKeys ni le Touch Ring via HID — ils restent inaccessibles sans accès USB direct (IOUSBLib).

Configuration

Fichier JSON dans ~/Library/Application Support/wacomd/config.json. Le démon le recharge à la volée sur réception de SIGHUP, envoyé automatiquement par l'app de paramétrage à chaque modification. Pas besoin de redémarrer le service.

Compatibilité

Système

  • macOS 26 Tahoe — testé
  • macOS 15 Sequoia — devrait fonctionner (API publiques inchangées)
  • macOS 14 Sonoma — idem
  • Apple Silicon + Intel

Tablettes

  • Wacom Intuos Pro Small — PTH-451 (VID 0x056a / PID 0x0314) ✓ testé
  • Famille Intuos5 / Pro — non testé mais format de rapport identique sous Linux, ajout 1-ligne dans KnownModels.swift
  • Modèles Bluetooth — non supporté pour l'instant
